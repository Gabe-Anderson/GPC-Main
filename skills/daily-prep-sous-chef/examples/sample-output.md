# Sample Output

This is intentionally short. The point is not to prove the agent scanned everything; the point is to show what it handled and what decision remains.

```text
The main thing: your 4 PM investor call is missing the updated deck link, but I found the latest version in Drive and prepared a one-paragraph agenda.

Handled
- Drafted the reply to a priority contact and placed it in the existing follow-up task.
- Checked tonight's dinner logistics: venue is confirmed, reservation is not. The restaurant uses Resy and has 7:30 and 8:00 available for 2.
- Moved the loose onboarding task under the active product-launch parent task.
- Updated yesterday's intro loop to waiting because you already sent the email.

Needs approval
- Book dinner at 7:30 PM for 2? If yes, I'll use the prepared reservation path.
- Send the priority-contact follow-up draft?

Blocked
- I could not access the project repo, so I could not verify whether the latest demo checklist changed.
```

## Why This Is Good

- It leads with the highest-leverage issue.
- It says what was handled.
- It asks only concrete approvals.
- It names the blocked surface only because it affects useful prep.
- It does not summarize the whole calendar or inbox.
